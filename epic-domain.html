Important Note About TXT DNS Records

Since this site is hosted on GitHub Pages (gamrabadr.github.io), direct DNS TXT records cannot be added because GitHub controls the DNS for *.github.io subdomains.

For Epic Games verification, use the file-based verification method by placing the verification content in the .well-known/epic-domain-verification.txt file.

Steps to Complete Verification

  1. Copy your domain secret from the Epic Developer Portal
  2. Update the .well-known/epic-domain-verification.txt file with your secret
  3. Commit and push the changes to this repository
  4. Wait for GitHub Pages deployment to complete
  5. Click "Verify" in the Epic Developer Portal

Custom Domain Option

If Epic requires a TXT DNS record specifically, you can set up a custom domain:

  1. Register a custom domain
  2. Configure it as a custom domain for this GitHub Pages site
  3. Add the TXT record through your domain registrar's DNS settings
